# ⛪ Mapa de Missas e Confissões

Um projeto simples e útil para ajudar pessoas a encontrarem **igrejas próximas**, com horários de **missas** e **confissões** exibidos diretamente no mapa.

---

## 🌍 Funcionalidades

- 📍 Visualização de igrejas em um mapa interativo
- ⛪ Pins personalizados para melhor identificação
- 📖 Detalhes da paróquia ao clicar no marcador
- 🕊 Horários de missas
- 🙏 Horários de confissões
- ➕ Envio de novas paróquias via formulário


---

## 🚀 Tecnologias utilizadas

- React
- TypeScript
- pigeon-maps
- CSS puro
